                          为什么要记录？
                          借用知乎上某位网友对问题“语言和文字有本质上的区别吗？”的回答：

                          “文字又是对语言的记录。是为了从时间上，把过去的思维记录下来。所以本质上看文字是语言的扩张，是又一次信息传递的生产力革命。通过文字对语言的记录，使得瞬间的思维得以固定下来，这带来更多的经验。”

                          “文字操纵思想，思想操控文字。”
