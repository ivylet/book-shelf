# 动态SQL

MyBatis通过采用功能强大的基于对象导航图语言的表达式来完成动态SQL，开发人员可以通过动态SQL灵活组装SQL语句，一定程度上避免了单一SQL语句的反复堆砌，提高了SQL语句的复用性。

## 基本元素

| 元素                           | 说明                         |
| ------------------------------ | ---------------------------- |
| `<if>`                         | 判断语句，用于单条件判断     |
| `<choose>(<when>,<otherwise>)` | 判断语句，用于多条件判断     |
| `<where>`                      | 简化SQL语句中where的条件判断 |
| `<trim>`                       |                              |
| `<set>`                        |                              |
| `<foreach>`                    |                              |
