# CSS简介

## CSS选择器

### CSS作用：

举例：

```html
<div>我是div</div>
<div>我是div</div>
<p>我是段落</p>
<ul>
	<li>我是ul里面小li哦</li>
</ul>
<ol>
	<li>我是ol里面小li哦</li>
</ol>
```

将div中文字改成红色，把第一个div里面的文字改为红色，把ul里面的li文字改成红色，要怎么操作？

### 选择器分类

选择器分为基础选择器和复合选择器，首先讲基础选择器：

基础选择器是单个选择器组成的，包括*标签选择器*、*类选择器*、*id选择器*和*通配符选择器*。

### 标签选择器

用HTML标签作为选择器，按标签名称分类，为页面中某一类标签制定统一的CSS样式。

语法：

```css
标签名{
    属性1：属性值1；
    属性2：属性值2；
    属性3：属性值3；
    ...
}
```

标签选择器可以将某一类标签全部选择出来，比如全部的<div>和所有的<span>可以快速为页面中同类型的标签统一样式，不能差异化样式，只能选择全部当前标签。

### 类选择器

差异化选择不同标签，单独选一个或者某几个标签，可以使用类选择器

语法：

```css
.类名{
    属性1：属性值1；
    ...
}
```

如：将所有拥有red类的HTML元素均为红色 

```css
.red{
	color:red;
}
```

结构需要用class属性来调用class类的意思

```html
<div class='red'>变红色</div>
```

类选择器需要用 "." 进行标记，后紧跟类名，相当于给该标签起名字，不能使用纯数字、中文，尽量使用英文字母，命名要规范。

### ID选择器

ID选择器可以标有特定id的HTML元素指定特定的样式。

HTML元素以ID属性来设置ID选择齐，CSS中ID选择器以"#"来定义

语法：

```css
#id名{
    属性1: 属性值1:
    ...
}
```

例如，将ID为nav元素中的内容设置为红色。

```css
#nav{
	color:red;
}
```

注：id属性只能在每个HTML文档中出现一次。

### ID选择器和类选择器的区别

ID选择器只能选择唯一的元素，类选择器可以选择很多个，最大的区别就是使用次数，类选择器多用于修改样式，ID选择器多用于修改页面唯一性的元素上，常和JAVAScript搭配使用。

### 通配符选择器

在CSS中，通配符选择器用 "*" 定义，用于选取页面中所有元素（标签）。

语法：

```css
*{
	属性1: 属性值1;
	...
}
```

通配符选择器不需要调用，自动给所有元素使用样式，特殊情况才用。

```css
*{
	margin: 0;
	padding: 0;
}
```
